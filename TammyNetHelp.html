How to configure your test scripts

  • In the Home Page click on New Configuration to create your own tests
  • Enter the Configuration name
  • Select what set up you want from the options (Tammy1, Tammy2, Tammy SIAM)
  • Click Next
  • Select the product (Orion, Fleet, TR100)
  • Click Next
  • Select required TA scripts
  • Click Next
  • Select Mode(UDP,TCP,CLI,SMS)
  • Click Review
  • Click Create and refresh your page
